This is just an example page to showcase how to use the Genesys dice font.

Step 1: Add Genesys Font

Use the .genesys class to access the font:

The dice shapes

  • j: j
  • k: k
  • l: l

The dice results

  • a: a
  • s: s
  • t: t
  • h: h
  • f: f
  • d: d

Power Level icons

  • c: c
  • p: p
  • g: g

Step 2: Add the Colour

The dice require not only the .dice class to add the outline, you also need to add the dice-name class to add the inner colour.

  • .boost: j
  • .ability: k
  • .proficiency: l
  • .setback: j
  • .difficulty: k
  • .challenge: l

One last thing, though. For accessability, be sure to use the aria-label in every span to make sure that screenreaders read what the symbol is and not just the letter.

For example: <span class="genesys" aria-label="two success">ss</span>
